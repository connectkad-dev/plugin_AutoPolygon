Création automatique de polygone via un point
Quand vous faites un clic gauche, un rectangle est créé avec le point bas gauche du rectangle comme référence par défaut. (Cette référence peut être changé dans les options) Si vous laissez le clic gauche enfoncé, vous pouvez tourner le rectangle.
Création automatique de polygone via une ligne
Quand vous faites un clic gauche, une ligne de rectangle est créée. Cette ligne ne peut être défini que par deux points. Pour valider la ligne de rectangles, vous devez presser le clic droit. Si vous laissez le clic droit enfoncé, vous pouvez choisir l'orientation du rectangle en bougeant le curseur de la souris à droite ou gauche de la ligne.
Options
Vous pouvez choisir la largeur, la longueur et la position de référence.
